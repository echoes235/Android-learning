# SunnyWeather

A simple Android weather application built with Kotlin.

## Features

- Display weather information including:
  - City name
  - Current temperature
  - Weather description
  - Humidity
  - Wind speed

## Project Structure

- `MainActivity.kt` - Main activity that displays weather information
- `model/Weather.kt` - Data model for weather information
- `layout/activity_main.xml` - UI layout for the main screen

## Requirements

- Android SDK 24 or higher
- Kotlin 1.9.10 or higher
- Android Gradle Plugin 8.1.2 or higher

## Building the Project

1. Open the project in Android Studio
2. Sync Gradle files
3. Run the project on an emulator or physical device

## Testing

The project includes unit tests for the Weather data model. Run tests with:

```bash
./gradlew test
```
